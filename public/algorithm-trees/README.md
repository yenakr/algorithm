# 원본 알고리즘 트리 이미지 업로드 가이드

이 폴더에 교수님께 받은 원본 알고리즘 트리 이미지를 업로드해 주세요. 

## 권장 파일명 및 매칭 관계

* **배설돌봄 알고리즘 트리 원본**: `toileting-original.png`
* **식사돌봄 알고리즘 트리 원본**: `feeding-original.png`
* **이승돌봄 알고리즘 트리 원본**: `transfer-original.png`

## 중요 권장 사항

1. **파일 형식**: 배경이 깔끔하게 보이는 **PNG** 형식을 가장 권장하며, 고화질 **JPG** 또는 최적화된 **WEBP** 형식도 가능합니다. (만약 다른 형식으로 업로드할 경우 확장자명을 `png`로 통일하거나 `src/data/index.ts` 파일에서 경로의 확장자 부분을 변경해 주시면 됩니다.)
2. **해상도 및 압축**:
   - 권장 가로 해상도는 **1200px ~ 2000px**입니다. 너무 작으면 글씨가 깨져 보일 수 있습니다.
   - 이미지가 너무 커서 로딩이 느린 경우 (예: 5MB 이상), [TinyPNG](https://tinypng.com/) 등의 온라인 이미지 압축 도구를 사용하여 용량을 1MB 이하로 압축하여 저장해 주세요.
